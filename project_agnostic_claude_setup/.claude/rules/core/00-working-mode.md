# Working Mode

Classify the task before choosing process.

- `small`: direct execution plus self-check
- `standard`: short plan, implementation, verification
- `deep`: requirements, plan, review, execution loop, learning capture

Lean by default. Escalate ceremony only when ambiguity or risk justifies it.

Apply the simplification filter before adding workflow:

1. Question the requirement.
2. Delete what is unnecessary.
3. Simplify before optimizing.
4. Accelerate only what still matters.
5. Automate only after the process is correct.
